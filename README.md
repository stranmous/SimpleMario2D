# 🎮 2D Game Project

A small 2D game project made while learning game development. This project was built to explore the fundamentals of game design, mechanics, and programming in a hands-on way.

## 🚀 Overview

This game is a beginner-friendly project focused on implementing core gameplay elements such as player movement, interactions, and basic game logic. It serves as a practical learning experience rather than a fully polished product.

## 🎯 Features

* Basic 2D player movement
* Core game mechanics implementation
* Simple game loop and controls
* Collision handling (if applicable)
* Minimal UI elements (score, health, etc., if included)

## 🛠️ Tech Stack

* Language: *GDScript*
* Framework / Library: *Godot Game Engine*
* Platform: *Desktop*

## 📚 What I Learned

* Fundamentals of game loops and rendering
* Handling user input and controls
* Managing game state and logic
* Debugging and improving performance
* Structuring a small-scale game project

## 📌 Future Improvements

* Add more levels or challenges
* Improve graphics and animations
* Introduce sound effects and music
* Optimize performance
* Refactor code for better structure
